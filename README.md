
# Meu sistema bancario - Formação Programador

O objetivo do projeto e treinar os conhecimentos ao longo da Formação Programador. durante o modulo 03. utilizando estruturas de repetiçoes e estruturas de dados para criar im script em Python que possibilite aa simulação de operaçoes bancarias do dia a dia.

### Funcionalidades 

Para isso, tomamos como base os seguintes requisitos/operações:

0. Um menu com seleção de operação
1. Criação de contas 
2. Remoção de contas 
3. Listagem de todas as contas criadas 
4. Adicionar saldo
5. Remover saldo
6. Transferir valor entre contas 
7. Consultar saldo de uma conta especifica 

### Estruturas de dados

As contas devem ser armazenadas em um lista e cada conta deve ser um dicionario ( chave-valor) contendo as seguintes chaves:

-numero_conta
-saldo_conta
